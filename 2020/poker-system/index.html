phase 0.

  • 最小级别桌：导致很多技巧受限，其他的一些因素重要性增强
  • 游戏目的：锻炼人性优于盈利
  • 先验性：非试错游戏，How you think about the game will determine how you play不能为了玩而玩、为了凭运气赶水而玩。
  • 数据号：隐藏、弥补我蹲牌风格

phase 1.价值下注

  • 人性之自律：蹲牌（手牌、pos）、弃牌、收集信息
  • 人性之自知：避免深筹（>2个买入，离桌）
  • 玩家研究：复盘研究对手打法特点，用于读牌拿价值
