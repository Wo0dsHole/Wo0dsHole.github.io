15-金钱陷阱

Written by Wo0dsHole with ♥ on October 1, 2019 in

金钱陷阱

读书时对金钱并没有多少思考，但仍可回忆出，那时心中隐隐的认知，是一种带着书生意气的轻视。

在毕业的这 3 年里，触发我对金钱的专门思考有 3 次，其中两次的感受重叠了。

因而在此，记下两次对金钱的思考，作为自己的态度。

0.涨薪与消费

大约一年半前，吃鸡崛起，成为一个现象级的游戏。而与之相伴的，自然是当下盛行的直播行业。

起初我对看直播很是不可理解——自己不玩游戏，看别人玩，而且很多主播还不是高手，意义何在呢？后来在某处看到有人将 “直播” 与 “看电视剧” 同等视之，觉得有道理，便开始接受看直播了。

大概一周之后，留在了一个固定的直播间。主播是一个退役的男职业选手，但身上没有很多因辍学走上游戏道路的人的轻狂浮躁，更像我搞技术的同事，给人以诚实可靠的感觉。

几月之后，又是在某处看到一句话 “看直播刷礼物，是在交荷尔蒙税”。顿觉这句话很有意思，想亲身体验一下，于是便有了我第一次送礼物看直播。但其实消费也并没多少，两三个月下来也就一千而已。不过看账单时还是有点惊讶，因为这些消费很隐形。

这时出现了我唯一一次工资调整，本来说是涨 3k，但最后又补充说明是算上五险一金后。简而言之，到手涨了 2k 出头吧。

可能是习惯了一发工资就存入余额宝，而自己又不怎么花钱。因此这样的收入变化，便有了一个数字上的直观感受。

而另一个有趣的感受出现了——之后在一次送几块钱的小礼物时，突然预算了一下一月下来的累积（未调薪前也算过）。自然还是几百而已，但由于我会做饭便又进一步将两者的开销比较。可这种比较其实是没有可比性的，虚拟与现实，只会根据个人的生活习惯总是偏向一方。这时一个未有过的有趣感受便出现了——觉得不值得。

这样的情绪转变引起了我的注意，尤其它还是被自己向来不看重的金钱引来的。

一番思考之后，我发现舍不得消费的根源，需求增加导致金钱的缺乏，而是金钱的增长本身！——小小的几千块收入变化，引发了我对金钱的欲求。

明白这一点后，我对自己有些失望。

因为我老早就想明白了一件事：对金钱的追求是合理的，正如 “君子爱财，取之有道”。但这个过程中存在的 “陷阱” 在于观念变成 “为了钱而去追求钱”。而这种转变内藏的因果是一个无休止的死循环。如果陷入了这个死循环，那么人的生活就被对钱的欲望所驱使。

反观这一次涨薪与消费态度的变化，从预知了我将来面对更大额的金钱追求时，很可能落入上面的陷阱。

为了改变这种命运，我必须思考 “需要多少钱？”、“金钱的积累方式？”。这样才能在金钱积累时，不至于耗费过多心力；在财富积累到一定程度时，不至于迷失在一种追求金钱的惯性里。

1.How much？

最早的一次触动对金钱的思考，其实正是和 “量” 相关的。
