0-为何写作

Written by Wo0dsHole with ♥ on October 31, 2019 in

文学类写作

  • 哲学、严肃文学、网络小说

“任何一个作家，都应该是一个哲学家。”

“书，是工具，也只是工具。”

三种不同的文体，不是三个不同的领域，而是层次关系。但根据读者态度不同、自我需求不同，演绎成了不同的文字形式。

哲学，是自我人生中不断的感受、感悟、思考。它能像一本日记随笔，记录着一个人一生中点点滴滴的思想痕迹。而它的读者主要是自己，所用的语言可能会缺少一些心里未写的 “背景” 信息，自己能意会。但若真让别的读者阅读，总会费心思琢磨，甚至先行了解作者自传。希望从经历、思想上不全那些未注解的 “背景”。

严肃文学，是自我思想中最成熟部分的布道。用故事、大量的文字描写，来填平读者与作者的个体思想差异。从而使得读者可以领会作者的思想，将这种思想变为自己的 “工具”，像是轮子，载着自己继续在人生路上前行。

网络小说，是自我的娱乐和物质的追求，也可能是我的 “避难所”。读者一目十行的态度，使得网络小说难以讲述清自己复杂的思想。那倒不如自娱自乐地将思想嵌入其中，同时以细腻精彩的文笔、虚幻的故事向读者征收 “七情六欲税”。也是 “君子爱财，取之有道” 的一种途径。

专业类写作

见 “方法论” 第 0 篇，科学是证伪的过程，学知识最快的方式是建立自己的认知模型，然后不断的改善。

附带的，给出一个简明的 “知识工具”，而不是在有限的知识上，故意混淆人眼界，让后来者难以进步。

思想基因

人的最终死亡是被人忘记。这曾是我建立博客的初衷——“这里，刻着我的墓志铭”。

但后来在 “free solo” 时，我体现到了另一种人生状态——人真正的孤独。

因而我现在并不觉得非要 “对同类友好” 地留下些什么。但把自己的思想传播下去，也不失为一件值得做的事，即使是它们会被剥去个人的标签。
