参考

其实没有太多新内容了，量的差异。由此可见对语言特性掌握的重要，即使Javascript这么臃肿的语言看完也没多少新东西。也一如 sicp 从复杂度控制出发对语言的观点：基本元素-组合方式-抽象方式。相比之下，感觉 js 弄了很多绕弯的概念，却既没有 Scheme 表达力强，也没有 Scheme 的清晰。

之后对于 Javascript 的语法细节，就放到读 ECMAScript 上时了，这里的字典版参考，既不非常的详细，又看不到更多的语言细节，没必要逐一翻看。

这几天通过 MDN 对 Javascript 的体系学习，获益良多——去魅的同时，对大型语言有了新的认识，以及 sicp 观点对语言学习的简洁性。不过之后要专门针对语言特性学习，这是更直接彻底的掌握所有语言之道。
