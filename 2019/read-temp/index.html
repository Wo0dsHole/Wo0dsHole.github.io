简记

《百年法》

文学，思维的试验。& 永生思考

《天才在左，疯子在右》

（客观）侧面

《平面国》

阶级嘲讽 & 何为未知

《月亮与六便士》

感触良多，没有一个单一的主题可概括

《面纱》

爱情的匹配

《生命不能承受之轻》

没看懂，感觉是《月亮与六便士》的难懂版，虽说有感触的地方不少，但越到后面越不明白了。尤其是小说的背景还和历史时期相关，这就像那些韩国高分电影，看得一脸蒙蔽，最后一搜评论，原来是有历史背景。但这并不是我喜欢的，更偏好《月亮与六便士》的风格。

《刀锋》

虽然故事完全不同，但看完后感觉是《月亮与六便士》中故事的另一个角度重写。《月亮与六便士》是从艺术、美，《刀锋》是从宗教、信仰。其中对于爱情，也有了自《月亮与六便士》、《面纱》更进一步的解释。

《且听风吟》

意识流。 从小书名听过很多次，没想到书极薄（150页）。先开始不知所云，中途突然想起高一语文老师说过我的一篇作文——意识流。猛然知道是什么意思了，原来这就是意识流。一种不需要情节的文学。

《人间失格》

不是我以为的，写一个“人”如何慢慢失去做人的资格。而是一个本来就没成为过“人”的人，如何慢慢失去被强行冠以的“人”的标签。 “有些人天生对这个世界不感兴趣，却无法选择地被生了下来。他们看了看这个世界，发现没什么可活的，于是选择了离开。”

《远山淡影》

本来真以为是用写的很淡的方式去刻画日本战后的改变，但最后才发现被作者的叙述方式惊了。可惜被译者的后记剧透了。不然肯定会读第二遍来解惑。

原来书中有些恐怖的一对母女其实是传统主角极端现实的“提炼”。越到后面越开始“重合”，以为作者或是翻译弄错了，直到最后的完全重叠，才猛然知道是同一个人。而这本书也不一定是在讲日本战后，而是“回忆”，回忆是最被扭曲的。

《老人与海》

真实的细腻、意志与肉体。 但此外还有一个疑问，《老人与海》早已被一句“人可以被毁灭，但不能被打败。”给喧宾夺主的注解了。但是，在书的最后——老人和最后一波鲨鱼搏斗之后，却说自己被打败了。我不明白这是为何？

（耶稣受难意象）

《围城》

之前一直看外国的译著，在语言方面只有《月亮与六便士》给过我辛辣之感。《围城》算是看的第一本中文书，被它“过分”写作语言惊了。

我以前只觉得写作要从侧面表达，这种侧面甚至不是对本体的比喻。而《围城》恰恰反之，它每一处语言都是那么直接，配大段的类比、比喻。但偏偏每一个都是那么贴切到位，甚至可以说是旁人绞尽脑汁才找得出的比喻。而这样的写法通篇都是，真的让人叹服。

我不知道文笔是什么，但这绝对是一种体现。尽管如此，但始终觉得和自己想要的不合，只承认它确实是值得反复读的“有趣玩物”。

就如之前读《且听风吟》明白了什么是意识流，对于意识流的书，甚至阅读这个动作本身带来的感觉也算是书的一部分，但这样的趣味比起《围城》来的太隔靴挠痒了。

但是读到最后几十页，方孙二人吵架又和好的日常时，让我有了慰藉。何时我也能和人吵架又坦然的和好，那么我人性中“爱”的孱弱便是痊愈了。

后记中杨绛写的钱钟书其人，又让我想到了关于“爱的匹配”，因为正好前几天写的“第8篇”里有提到钱杨的爱情。此时了解到钱钟书其人后，我觉得他能获得那样一份爱情是合理的，一如他的国文水平也是如此。

主观上的偏执是可以超过客观上的天分的。但这种偏执我却不敢去尝试，怕那是一生。也许我该去看看《革命之路》，想想平凡和不凡。这是我看了《年轻的心在哭泣》第一部时联想到的，可能和书不契合。

《年轻的心在哭泣》

一幕幕“著书不立说”的人生展示。 给人预见、焦虑，但自行思考、解决。

《赴宴之前》

感觉不怎么精彩，后面几篇有些东西的，但是因为之前看了毛姆的基本代表作，又让我有一种读它们的导读的感觉。

《我的职业是小说家》

真正的 “职业” 角度，有参考意义。

《天蓝色的彼岸》

“你准备死多久？”但是比《百年法》的 “你准备活多久？”精彩、有深度多了。

《局外人》

与《杀死一只知更鸟》所讲的 “偏见” 有类似之处，但是更文学、哲学化，更像是一场思维的实验。最后长长多页的内心独白，太亮了，像是一段 solo。

《了不起的盖茨比》

金钱至上的美国梦时代，梦的破碎。结合作者自身的背景经历，但我更喜欢《一千零一夜》中的点评，与书中时期类似的现在中国，是否应该想想中国梦的破碎呢？

说实话，我觉得中国梦一词有些跟风，而且我甚至都不知道中国梦的内容是啥。这就挺荒诞了。

作者的文笔却是华丽，让我有一些读《围城》时的感觉，里面的那些妙意的比喻。

《杀死一只知更鸟》

  1. 孩子是会“为某些人给其他人带来的苦难而哭泣”的人。
  2. 但要记住，杀死一只知更鸟便是犯罪。 “知更鸟只是哼唱美妙的音乐供人们欣赏，什么坏事也不做。 它们不吃人家院子里种的花果蔬菜， 也不在谷仓里筑巢做窝， 只是为我们尽情地唱歌。 所以说杀死一只知更鸟是犯罪。”
  3. 迫害，都是来自那些怀有偏见的人。
  4. 你永远也不可能真正了解一个人， 除非你穿上他的鞋子走来走去， 站在他的角度考虑问题。 对我来说，站在拉德利家的前廊上就足够了。

“善良，是一种选择。”

我不知道有多少人，只是一如被鸡汤激起的情绪，便不加思索地匆匆对这句话点赞。

而另一个不被选择的对象是什么呢？ 人性的暗面。

真正“选择善良”的人，总会有一种对自我怀疑的愁苦。

因为他看见了内心的善良周围，趴着的是一圈阴暗。 知道原来自己的本质并非纯粹， 也就对这种“选择的善良”产生是否跟着“虚荣”的影子的怀疑？

然而，我在高中时的一次邂逅，让我惊艳地发现，原来是存在“纯粹善良”的人。 “斯人若彩虹，遇上方知有。”贴切地描述了那一刻，我心中的惊艳感。

而这样的人就是书中的“知更鸟”。

书中杀死“知更鸟”的武器，是偏见。 一种是肤色偏见，一种是乌合之众对保持自我独立生活方式的人的偏见。

前者已经杀死了一只“知更鸟”，后者被扼杀于萌芽，挽救了一只“知更鸟”。

书中所讲的东西很明显，对偏见的控诉。毋庸赘言。

ps: 人生建议：“选择善良”的人，承认人性的复杂，是对自我的善待。（感于《月亮与六便士》）

《看不见的城市》

这是目前看过的，最为精巧的一本书。

短短160页，而且都是一两页单独成一篇，却让我看了很久。相比150页的《且听风吟》，虽然是意识流，却也就两三小时读完。

这本书更像是诗歌，每一篇里的城市都是一个 “意象”，可以让读的人延伸出不同思考，而系列的标题有给这些思考以提示。

“作者只有作品本身有价值，因而我不提供任何传记资料。”我很喜欢作者这种观念，它让书本身真正成为了一个工具，载着读者前进，而不是以自身的隐含背景，将读者困在原地。

以后重读写感受时，应该会和这本书相等篇幅。

卡尔维诺，第一次让我感到天赋超然的作家。相信他的短篇小说集一定比毛姆精彩。

《分成两半的子爵》

按作者后记中的话，此书是关于如何自我实现中 “追求不受社会摧残的人生”。

书中子爵的残缺是显式的，而映射的现代人的残缺则是隐式的。这种残缺体现在人单一的 “主义化” 人生状态，比如享乐主义、颓废主义、道德主义……

通过书中不同情节里这些人物的不合理行为，放大出这些残缺类型各自的缺陷，以此表明完整人性的追求。

这种完整人性，一方面是对人性复杂面的包容（如善恶），另一方面是有自己刻意的引导（如后记所言，“一个人心甘情愿的给自己立下一条规矩，没有这条规矩他将不是他自己。”）

这种观点是一种 “补全”，隐含着 自身现有的人性 “确实属于自己” 的前提基础。但如果从社会学中的 “习得性” 来看，人之初，性无善恶，都是后天学习而来。

那么问题来了，现在自己所拥有的人性，是否只是曾经从别处（如家庭、教育）获得的一条条 “木偶牵线” 呢？

正如吃下去的食物快速更迭着肉体，曾经被输入的思想是否也构成了现在的 “我” 呢？

如此一来，书中观点的基石便垮塌了——现在的自身只是别人思想的延续，没有自我又何谈完整与否？

但其实上面提到的后记中，“立下规矩” 近似于我认同的解答，但却没有说出其背后的深层原由。（其实是讨论 “忒修斯悖论”）

正如现代科学建立于一些不需证明的 “公理” 或是假设 之上，对自我存在性的解答也需要这样一个 “基点”。这个基点就是相信 “人” 的天赋是 “智慧”。

曾经别人思想的输入可以堆成现在自身的思想，但是这个过程中产生的智慧却是自己的。那么通过智慧对自身思想的源头进行审查、选择、重建，就可以斩断这些 “木偶牵线”，成为一个真正的 “自我”。

至于这个基点是否可靠，也许又是另一个深奥的哲学问题了。但至少现在可以建立起自我了，就如同现在所有的科学。

《再见，吾爱》

觉得有趣的倒不是故事本身，而是作者写作的方式。

有些说作者的语言洗练，但我感触更深的，是他有一种讲故事人的自娱感。

总能在书中感受到作者的存在，很多时候似乎他在跟你说笑一般。不知道我以后的网络小说能否加入这种风格。
