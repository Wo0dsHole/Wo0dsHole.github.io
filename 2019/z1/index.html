裸辞尾声

Written by Wo0dsHole with ♥ on September 22, 2019 in

写了一个通宵的《裸辞思考》，心中清明了起来。但最后得到的结果，却好像和之前有次差不多。

人生和时间一样，是有方向的。而每天的重复又像是递归，想要发挥递归的强大能力，必须编写好每天之中的 “情况分支”，之后就只是简单的朝着各自的方向前行。

《裸辞思考》中我定下了方向，也是确定了每天的 “情况分支”。这里的旁白，就是简单的安排规划一下具体事情。

  1. 时间的管理：作息的固定（8h）、各项事情投入时间长度的保证
  2. 饮食结构的控制（1h）
  3. 阅读：1h
  4. 写作：2h
  5. 考公：2h
  6. 工作积累：4h
  7. 其他：6h

大致事项：

  • 管理时间：all
  • 尝试转行的准备：<=国庆
  • 入职，转行失败则投原行业：10月
  • 阅读习惯：all
